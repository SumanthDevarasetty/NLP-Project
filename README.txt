To test the LSA code/ESA code/NESA code, run LSA.py/ESA.py/NESA.py with the appropriate arguments.
Usage: LSA.py [-custom] [-dataset DATASET FOLDER] [-out_folder OUTPUT FOLDER]

A plot for various evaluation measures @k=1,..10 is generated when -custom flag is not used
            
When the -custom flag is passed, the system will take a query from the user as input. For example:
> python LSA.py -custom
> Enter query below
